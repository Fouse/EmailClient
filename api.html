@730 is an email client similar to microsoft outlook, built with the purpose of managing emails. There is a separate module that implements the @730 Email Client functionality. It can be accessed through a global variable named mailapp. Inside this global variable we have the following functions available:
Function Name Parameters Return Value Description
Folder name: refers to the name of the folder None This function assign name to a folder throught its parameter name, and store emails.
Email sender, receiver, subject, message: email attribute None This function assign email's attributes.
createFolder name: name of the folder None This function allow you to create a folder, and name it.
addEmail folderName, sender, receiver, subject, message None This function add email to a specific folder with all of its attributes.
getFolder folderName email This function return email from a folder.
